# Auth API (Node.js)

A secure authentication & authorization API built with Node.js, Express, JWT, and MongoDB.

---

## Features

- User registration & login (bcrypt)
- JWT authentication (access & refresh tokens)
- Protected routes
- Role-based authorization (RBAC)
- Token refresh & logout

---

## Tech Stack

Node.js • Express • MongoDB • Mongoose • JWT 

---

## Setup

```bash
npm install
npm run dev

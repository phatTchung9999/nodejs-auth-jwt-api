# Full Stack Authentication App

This project is a full-stack web application with separate client and backend folders.

## Project Structure

### client/
The client folder contains the front-end application built with React.

Responsibilities:
- User interface (UI)
- Routing with React Router
- Login form handling
- Authentication flow
- Fetching and displaying data from the backend API
- Sending API requests to the backend

### backend/
The backend folder contains the server-side application built with Node.js and Express.

Responsibilities:
- JWT authentication
- API routes
- User authentication and authorization
- Data management
- Serving application data to the client

## Technologies Used

### Front-End
- React
- React Router
- JavaScript
- CSS

### Back-End
- Node.js
- Express.js
- JWT (JSON Web Token)
- bcrypt
- REST API

## Features
- User login system
- Protected routes
- JWT authentication
- API communication
- Dynamic data fetching and rendering
- Front-end and back-end separation
